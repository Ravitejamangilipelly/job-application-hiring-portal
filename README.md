## Job Application & Hiring Portal
Overview:
A simple job application platform built with React.js (frontend), Node.js + Express (backend), and PostgreSQL.

Admin: Create, edit, delete jobs, view applicants.
Candidate / User: Browse jobs, apply, save jobs, view applications & favourites.


## Tech Stack Used
## Frontend
React.js – UI library for building interactive user interfaces

React Router v6 – Client-side routing

Axios – HTTP client to communicate with backend APIs

CSS – Styling

## Backend
Node.js – JavaScript runtime
Express.js – Web framework for building APIs
PostgreSQL – Relational database
pg – PostgreSQL client for Node.js
bcrypt – Password hashing
jsonwebtoken (JWT) – Authentication
express-validator – Request validation
Other Tools:
nodemon – Auto-reload server during development
dotenv – Environment variable management

## Features
JWT authentication
Role-based access (Admin / Candidate)
Admin: CRUD jobs, view applicants
Candidate: Browse jobs, apply, save, view applications/favourites

# Getting Started with Create React App
Installation:
Clone the repository:
git clone <repo-url>
cd frontend   # or backend
Install dependencies
npm install
Run the frontend:
npm start
Run the backend:
npm run dev   # or npm start
